		数学知识：
		角度和弧度的区别：
			角度：两条射线从圆心向圆周射出，形成一个夹角和夹角正对的一段弧。当这段弧长正好等于圆周长的360分之一时，两条射线的夹角的大小为1度。
			弧度：两条射线从圆心向圆周射出，形成一个夹角和夹角正对的一段弧。当这段弧长正好等于圆的半径时，两条射线的夹角大小为1弧度。	
		有定义可知：圆所对应的弧长是半径的几倍，那么角的大小就是几弧度。
				   周角(360度)= 2π弧度
				   平角(180度)= π弧度
